What is XML?

  • XML stands for eXtensible Markup Language.
  • It encodes documents in a format readable by both humans and machines.

How is XML used?

  • Storage and transport of data.
  • Commonly used in web services and data interchange between systems.

What is JavaScript?

  • A programming language for adding interactivity and dynamic behavior to web pages.

How is JavaScript related to HTML and CSS?

  • HTML structures content.
  • CSS styles content.
  • JavaScript adds interactivity and behavior.

JavaScript Elements Example

  • Variables: Used to store data values. 
    let greeting = 'Hello, World!';
  • Properties: length: A property that returns the length of a string. 
    
let message = 'This is a message.';
let lengthOfMessage = message.length;
  • Methods: toUpperCase(): A method that converts a string to uppercase. 
    
let lowercase = 'convert me to uppercase';
let uppercase = lowercase.toUpperCase();
  • Functions: console.log(): A function used to log data to the console. 
    
let num = 10;
console.log('The value of num is: ' + num);
